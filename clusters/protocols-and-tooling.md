# Protocols and Tooling

Protocols and tooling terms describe how agent systems connect to external
capabilities, data, and applications.

## Terms

- [Tool Calling](../terms/tool-calling.md)
- [MCP](../terms/mcp.md)
- [Orchestrator](../terms/orchestrator.md)
- [Trace](../terms/trace.md)

## Useful Distinction

A tool is a capability an agent can use. A protocol defines how systems expose
and call those capabilities. An orchestrator decides when and how pieces of the
system should be used together.
