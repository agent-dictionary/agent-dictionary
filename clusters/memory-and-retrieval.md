# Memory and Retrieval

Memory and retrieval terms describe how an agent system keeps useful context
available across a task, a session, or many sessions.

## Terms

- [Memory](../terms/memory.md)
- [Context Window](../terms/context-window.md)
- [Retrieval](../terms/retrieval.md)
- [Trace](../terms/trace.md)
- [System Prompt](../terms/system-prompt.md)

## Useful Distinction

Context is what the model can currently see. Memory is stored information that
may be brought back later. Retrieval is one way to choose what memory or source
material enters the current context.
