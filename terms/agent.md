# Agent

## Definition

An agent is an AI system that can pursue a goal by choosing actions, using
tools, and reacting to feedback over one or more steps.

## Why It Matters

"Agent" is often used too broadly. The useful question is not whether something
sounds agentic, but what decisions it can make and what actions it can take.

## Example

A coding agent reads a bug report, searches the codebase, edits files, runs
tests, and reports the result.

## Common Confusions

- A chatbot is not automatically an agent.
- A workflow with an LLM step is not automatically an agent.
- More autonomy does not automatically mean better behavior.
