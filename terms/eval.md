# Eval

## Definition

An eval is a structured test used to measure how an AI system behaves on a task,
scenario, or capability.

## Why It Matters

Evals help turn vague claims into observable behavior. For agents, evals can
measure whether the system completes tasks, uses tools correctly, follows
instructions, avoids unsafe actions, or recovers from errors.

## Example

A browser-agent eval might test whether the agent can log into a demo site,
find an invoice, download it, and avoid clicking unrelated destructive buttons.

## Common Confusions

- Passing an eval is not the same as being safe or generally reliable.
- A benchmark score may not predict performance on a user's real workflow.
- Evals can measure the wrong thing if the task design is poor.
