# The Agent Dictionary

Plain-English definitions for the terminology around AI agents.

The goal is not to make vague agent language sound more impressive. The goal is
to make it easier to tell what a system actually does, where the human stays in
control, and which words are carrying real technical meaning.

This project is a work in progress.

## Entry Format

Each term page uses the same basic structure:

- a short definition
- why the term matters
- a concrete example
- common confusions
- links to related terms

Use [the term template](templates/term.md) for new entries.

## Contributing

See [CONTRIBUTING.md](CONTRIBUTING.md) before adding terms. The short version:
define terms in plain English, include examples, cite sources when a term depends
on a specific project or standard, and do not explain jargon with more jargon.
